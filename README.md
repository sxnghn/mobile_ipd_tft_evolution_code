# 1차원 이동형 반복 죄수의 딜레마에서 TFT 전략의 진화적 안정성과 임계 초기 비율

이 저장소는 아래 논문의 연구 및 결과 검증을 위해 사용된 공식 시뮬레이션 코드를 모은 것이다.

- **논문명:** 1차원 이동형 반복 죄수의 딜레마에서 TFT 전략의 진화적 안정성과 임계 초기 비율
- **저자:** 강성현, 정진희 (서울과학고등학교)
- **지도교사:** 노예솔

본 연구에서 제시한 1차원 연속 공간에서의 이동형 반복 죄수의 딜레마(IPD) 모형을 구현하고, TFT 전략의 진화적 안정성과 유한 크기 효과에 따른 임계 초기 비율($p_c$)을 분석하기 위해 작성되었다.

## 📂 코드 구성

1. **`ess_analysis`**
   - ESS(진화적 안정 전략) 분석 및 이론적 고정점($1/21 \approx 4.76\%$)을 확인하기 위한 코드.

2. **`twenty_strategies_sim`**
   - TFT와 ALLD를 포함하여 총 20대 전략 간의 상호작용, 경쟁 및 진화 과정을 시뮬레이션하는 코드.

3. **`critical_fraction_validation`**
   - 개체 수($N = 500 \sim 100,000$) 변화에 따른 유한 크기 효과를 측정하고, 임계 초기 비율($p_c$)의 상전이 현상을 검증하는 코드.
